Permite que tu asistente de IA programe reuniones y citas sin problemas mediante la integración con Cal.com. Esta potente función permite que tu IA consulte la disponibilidad y reserve reuniones directamente durante las llamadas.

<Warning>
  No configure ningún campo nuevo como obligatorio en la configuración de eventos de Cal.com. La integración solo admite los campos predeterminados y el campo de teléfono opcional. Agregar campos personalizados obligatorios provocará un error en la integración.
</Warning>

## Prerrequisitos

Antes de configurar la integración de Cal.com, necesitará:

- Una cuenta de Cal.com (alojada o autoalojada)
- Un tipo de evento configurado en Cal.com
- Una clave API de Cal.com

## Proceso de configuración

### 1. Configuración de Cal.com

1. Regístrese o inicie sesión en su cuenta [Cal.com](https://cal.com/) 
2. Cree un nuevo tipo de evento específicamente para reservas de asistentes de IA
   - Dale un nombre claro como "Reunión con Asistente de IA".
   - Configurar la duración, la disponibilidad y otros ajustes
3. Una vez creado, anote el ID del evento de la URL
   <img height="200" src="/resources/cal-event-id.png" />
4. Generar una clave API:
   - Vaya a la configuración de Cal.com
   - Vaya a Configuración del desarrollador
   - Crear una nueva clave API
   - Copiar y almacenar de forma segura la clave API

### 2. Información de contacto del cliente

#### Configuración de correo electrónico

Para enviar invitaciones de calendario a los clientes:

1. Define `email` en las variables de entrada de tu asistente
2. Pase el correo electrónico del cliente como `email` variable al crear el cliente potencial
3. El sistema utilizará automáticamente este correo electrónico para las invitaciones del calendario.

#### Número de teléfono en eventos del calendario

Para incluir el número de teléfono de la persona que llama en los eventos del calendario:

1. Vaya a la configuración de su evento en Cal.com
2. Haga clic en "Agregar una pregunta" 
   <img height="200" src="/resources/cal-event-settings.jpeg" />
3. Configure el campo con estos valores exactos:
   - Identificador: `phone` debe ser exacto)
     <img height="200" src="/resources/cal-phone-field.jpeg" />
4. Guarda tus cambios
   <img height="200" src="/resources/cal-save.jpeg" />

### 3. Cómo agregar Cal.com a tu Asistente

1. En [ la configuración de tu asistente de IA ](/ai-assistants/settings/prompt-and-tools#default-tools), ubica la sección Herramientas
2. Agregue la integración de Cal.com
3. Configurar la herramienta con: 
   - Su clave API de Cal.com
   - El ID del tipo de evento
     <img height="200" src="/resources/cal-tool.png" />

## Mejores prácticas

- Cree tipos de eventos dedicados para reservas de IA para rastrear su uso
- Establezca tiempos de amortiguación claros entre reuniones
- Configurar duraciones de reuniones adecuadas
- Utilice nombres de eventos descriptivos que tengan sentido para sus clientes
- Mantener solo los campos predeterminados y el campo de teléfono opcional
- Incluya siempre el correo electrónico del cliente en la creación de clientes potenciales para las invitaciones del calendario.

## Pruebas

Después de la configuración, recomendamos:

1. Realizar una llamada de prueba a su asistente de IA
2. Solicitar programar una reunión
3. Verificar que la reserva aparezca en su panel de Cal.com
4. Comprobar que el número de teléfono aparece en el evento del calendario
5. Verificar que el cliente recibe el correo electrónico de invitación del calendario
6. Comprobando que se han recibido los correos de confirmación

## Solución de problemas

### Soluciones rápidas:

**Error de clave API faltante:**
- **Solución:** Ingrese la clave API y el ID del evento correctos en la configuración del asistente
- **Pasos:** Vaya a  [la configuración del asistente ](/ai-assistants/settings/prompt-and-tools#default-tools) → Programación de citas → ngrese la clave API y el ID del evento → Guardar

**Errores de reserva:**
- **Se requiere correo electrónico:** asegúrese de que el asistente tenga el correo electrónico definido en [la configuración del asistente](/ai-assistants/settings/prompt-and-tools#call-variables)
- **Configuración del campo Teléfono:** Agregue el campo Teléfono en la configuración de eventos de Cal.com con el identificador "teléfono". Consulte [Número de teléfono en eventos del calendario"](#phone-number-in-calendar-events) para obtener información detallada.
- **Prueba con llamadas telefónicas reales:** utiliza una llamada telefónica (no una llamada web) para realizar la prueba
- **Sin campos personalizados:** no agregue campos personalizados obligatorios en la configuración de eventos de Cal.com

**Problemas de conexión::**
- **Verificar que la clave API** esté activa en la configuración de desarrollador de Cal.com
- **Verifique el ID del evento** desde la URL de su evento de Cal.com
- **Confirme que el identificador del campo de teléfono** sea exactamente "teléfono"

<Note>
Para obtener una resolución completa de problemas de Cal.com, incluidas soluciones detalladas y problemas adicionales, consulte [Problemas de Cal.com](/troubleshooting/cal-com).
</Note>
